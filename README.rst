Simple Flask Weather App
========================
   
Prosta aplikacja wyświetlająca aktualną pogodę w podanym mieście

- Tworzymy, aktywujemy i konfigurujemy wirtualne środowisko:

  virtualenv venv
  source venv/bin/activate
  make deps

- Uruchamianie aplikacji:

  make run

- Uruchamianie testów

  make test
